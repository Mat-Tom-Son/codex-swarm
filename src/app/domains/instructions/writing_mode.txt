5. Edit for clarity, coherence, and correctness

- Create and edit markdown documents
- Convert between document formats (using python-docx for Word, etc.)
- Reference and incorporate content from other documents
- Format citations and bibliographies

- Always cite sources properly
- Include file names or paths for traceability
- Maintain a clear distinction between original content and references
